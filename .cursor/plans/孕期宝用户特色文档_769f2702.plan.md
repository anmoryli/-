# 孕期宝用户特色功能文档

## 文档定位与结构

面向用户的功能介绍文档，突出产品特色与使用价值。建议输出为 Markdown 文件，可放在项目根目录或 `docs/` 下，如 `docs/用户功能说明.md` 或 `FEATURES.md`。

---

## 一、核心定位

- **应用名称**：孕期宝 - 把每一天都留下来
- **核心理念**：专注孕期记录，成为你和宝宝最珍贵的回忆
- **目标用户**：孕妇本人、配偶（爸爸）、其他家庭成员（婆婆、妈妈等）

---

## 二、功能模块清单（按代码梳理）

### 1. 账户与身份


| 功能   | 说明                                                  | 入口/路径              |
| ---- | --------------------------------------------------- | ------------------ |
| 注册   | 支持孕妇/家庭成员两种身份；孕妇需填写怀孕日、预产期；家庭成员可选关系（配偶、婆婆、妈妈、爸爸、其他） | `/register`        |
| 登录   | 用户名密码登录                                             | `/login`           |
| 忘记密码 | 找回密码流程                                              | `/forgot-password` |
| 欢迎页  | 新用户引导，介绍核心功能（记录、孕期小伴、小家、爸爸成长营、健康值、社区等）              | `/welcome`         |

### 3. 孕期记录（时光轴）


| 功能      | 说明                         | 入口              |
| ------- | -------------------------- | --------------- |
| 写日记     | 文字记录，支持心情、分类、可见范围          | 记录 → + → 写日记    |
| 拍照片     | 多图上传，可写描述                  | 记录 → + → 拍照片    |
| 录语音     | 录音并自动转文字保存                 | 记录 → + → 录语音    |
| 上传文件    | 文件类记录                      | 记录 → + → 上传文件   |
| 写信给宝宝   | 特殊标签，可设解锁时间（宝宝 3 岁等）       | 记录 → + → 写信给宝宝  |
| 给未来的自己  | 时光胶囊类记录                    | 记录 → + → 给未来的自己 |
| 筛选      | 全部/重要时刻/给未来的自己/文字/照片/语音/文件 | 记录页 Tab         |
| 日期筛选    | 按日期范围筛选                    | 记录页日期选择器        |
| 妈妈/爸爸记录 | 有配偶时可按记录人筛选                | 记录页 Tab         |
| 搜索      | 按标题、内容、标签搜索                | 记录页搜索框          |
| 导出 PDF  | 按日期范围、妈妈/爸爸/两人导出至邮箱        | 记录页 → 导出 PDF    |
| 日期范围分享  | 选定日期范围生成 PDF 分享            | 记录页日期筛选区        |
| 编辑/删除   | 本人或配偶可编辑、删除记录              | 记录卡片操作          |
